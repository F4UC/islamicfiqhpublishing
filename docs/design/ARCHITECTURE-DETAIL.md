## 1. นครวงกลม (Round City / Madinat al-Salam) — ผังรวม
- ★รูปวงกลมสมบูรณ์★ เส้นผ่านศูนย์กลาง ~2 กม. (บางบันทึกถึง 2,700 ม. / ~1,000+ ศอก) — อิงผังซาสซานิด (Gur/Firuzabad) + เรขาคณิตยุคลิด
- ก่อตั้ง 30 ก.ค. / 2 ส.ค. 762 · สถาปนิก 4 คน · เสร็จทั้งเมือง 766 · พระราชวัง+มัสยิดเสร็จ 763
- โครงสร้าง = ★วงแหวนซ้อน:★ วงนอก (ที่อยู่อาศัย/พาณิชย์) → วงใน (เพิ่มตอนสร้างจริง) → ★ลานกลางโล่งกว้างว่างเปล่า★ (esplanade) ที่มีพระราชวัง+มัสยิดอยู่แกนกลาง
- ★ถนนตรง 4 สาย★ จากประตูตัดเข้าศูนย์กลาง → แบ่งเมืองเป็น 4 ส่วน
- คูเมืองลึก (moat) ล้อมรอบ
- ★หมายเหตุ: เป็น "เมืองราชการ/ป้อม" ไม่ใช่เมืองตลาด★ — ตลาดถูกย้ายออกไป Karkh ปี 773 (เหตุผลความมั่นคง: ประตูจะได้ไม่ต้องเปิดกลางคืน)

## 2. กำแพง + คูเมือง (วิศวกรรมป้องกัน)
- ★กำแพง 2 ชั้น (double walls)★ — ฐานหนา ★44 ม.★ → เรียวเป็น ★12 ม.★ ที่ยอด · สูง ★~30 ม.★ (รวมใบเสมา)
- ★วัสดุกำแพง = อิฐดินดิบ (sun-dried mud brick) เสริมไม้อ้อ (reeds)★ · ★อิฐเผา (baked brick) ใช้เฉพาะส่วนโค้ง/โดม★
- ★อิฐมาตรฐาน ~18 นิ้ว (460 มม.) สี่เหลี่ยมจัตุรัส★
- ★ใบเสมา (merlons) เจาะช่องเชิงเทิน (embrasures)★ — สังเกตการณ์ + ยิงธนูตอบโต้
- ปริมาณอิฐ (บันทึก al-Khatib): 1/3 ความสูงชั้นแรกใช้อิฐ 162,000 ก้อน · ชั้น 2 = 150,000 · ชั้น 3 = 140,000
- แรงงาน ~100,000 คน · งบ 18 ล้านดินาร์ทอง / 100 ล้านดิรฮัมเงิน (บันทึก al-Tabari ที่ละเอียดกว่า = ~4.83 ล้านดิรฮัม เฉพาะโครงสร้างหลัก)

## 3. ประตูเมือง 4 ประตู (ดีเทลเต็มใน GATE-SPEC v1.2)
- ★4 ประตู:★ Kufa (ตต.เฉียงใต้ → เส้นทางแสวงบุญมักกะฮ์) · Basra (ตอ.เฉียงใต้) · Sham/Damascus (ตต.เฉียงเหนือ) · Khurasan (ตอ.เฉียงเหนือ → ถนนสายคุรอซานใหญ่)
- ★เป็น "หอประตู" (gatehouse) ไม่ใช่บานประตูแบน★ — ป้อมซ้อนชั้น มีทางหักงอ (bent entrance) + ห้องเหนือประตูมีโดม
- ★บานเหล็กคู่หนักมหาศาล★ สูงพอให้ทหารม้าถือธงควบผ่านโดยไม่ก้มหัว (ช่องเปิด ~6–8 ม.) · หนักจนต้องใช้คนหลายคนเปิด-ปิด
- (ตำนาน al-Tabari: บานเหล็กหล่อโดยญินให้กษัตริย์โซโลมอน — legend)
- รูปทรง: pishtaq (กรอบสี่เหลี่ยมยกสูง) + ซุ้มโค้งแหลม (pointed arch) เว้าลึก (iwan recess) + หอกลม 2 ข้าง batter + ใบเสมาขั้นบันได + บานไม้ตีเหล็กตอกหมุด
- ตกแต่ง: อิฐฮะซัรบาฟ + ปูนปั้นแกะ (Samarra beveled) + จารึกคูฟิก (นามธรรม) + ดาว 8/12 แฉก (girih)
> ★ดูดีเทลครบ + สเกล + วัสดุ PBR ใน GATE-SPEC.md v1.2★

## 4. ศูนย์กลาง: พระราชวังประตูทอง + โดมเขียว + มัสยิดหลวง
**พระราชวังประตูทอง (Qasr Bab al-Dhahab) = พระราชวังโดมเขียว:**
- ★โถงว่าราชการสี่เหลี่ยมจัตุรัส 30 ฟุต (9.1 ม.)★ เพดานโค้งสูง · เหนือขึ้นไป = ห้องขนาดเท่ากันใต้โดม
- ★โดมเขียว (Qubbat al-Khadra) สูง ~48 ม.★ · ★ยอดโดม = รูปปั้นนักรบขี่ม้าถือหอก★ (ตำนาน: หมุนชี้ทิศศัตรู — label "ตามตำนาน")
- ★อีวานหน้าโถง สูง 45 ฟุต (14 ม.) × กว้าง 30 ฟุต (9.1 ม.)★ — สไตล์ซาสซานิด
- โดมเขียวพังคืน 9 มี.ค. 941 (พายุฝน/ฟ้าผ่า) · ผนังล่างเหลือถึงมองโกล 1258

**มัสยิดหลวงอัลมันศูร:**
- ★hypostyle (เสาหลายแถว) หลังคาแบน สไตล์เปอร์เซีย (apadana)★ · เสร็จ 763
- ★ยุคแรกไม่มีหออะซาน (minaret)★ — เน้นลานกว้างรองรับละหมาดรวม
- โถงทั้งสอง (วัง+มัสยิด) = สัญลักษณ์อำนาจโลก+ธรรม รวมศูนย์

## 5. มัดราซาอัลมุสตันศิรียะฮ์ (Mustansiriya, 1227–1234, ฝั่งตอ.)
> ★สถาปัตยกรรมอับบาซิดของแท้ที่เหลือสมบูรณ์สุด★ — ใช้เป็นหลักฐานรูปธรรมได้
- ★ผังสี่เหลี่ยมผืนผ้า อิฐ 2 ชั้น · ผัง "สี่อีวาน" (four-iwan) ล้อมลานกลาง (courtyard) เปิดรับแสง★
- ★มหาวิทยาลัยแรกที่รวม 4 มัซฮับ (ฮะนะฟี/ชาฟิอี/มาลิกี/ฮัมบะลี) ไว้ที่เดียว — แต่ละมัซฮับประจำ 1 มุมอาคาร★
- มีดารุลฮะดีษ + ดารุลกุรอาน (+ดูแลเด็กกำพร้า) + โรงเรียนแพทย์มีโรงพยาบาลในตัว + ห้องสมุดใหญ่ (80,000–400,000 เล่ม) + หอพัก + ฮัมมัม + ห้องเสบียง
- ★นาฬิกาน้ำอนุสาวรีย์ (1235, Ali ibn Tha'lab):★ เหยี่ยวทองทิ้งลูกเหล็กลงชาม + ประตูทอง/เงินเปิดตามเวลา + จำลองการเคลื่อนของดวงดาว
- ★ทางเข้า = ซุ้มโค้งทรงกรวย 3 ซุ้ม★ + ลายดาว/โพลีกอน + อิฐสลักผสมอาราเบสก์ + ★เพดานมุก็อรนัส (muqarnas) รับน้ำหนักส่วนโค้ง★
- ★sustainable: ผนังอิฐหนา = thermal mass · ลานกลาง = ระบายอากาศ · พื้นที่เขียว+บ่อน้ำ = ลดอุณหภูมิ★

## 6. พระราชวังอับบาซิด / อัชชะรอบียะฮ์ (Abbasid Palace, ฝั่งตอ. al-Maidan)
> (ยังถกเถียงว่าเป็นวัง หรือมัดราซา — ผังคล้าย Mustansiriya)
- อิฐ 2 ชั้น · ★18 ห้องชั้นล่าง + 22 ห้องชั้นบน★ ล้อมลานกลาง
- ★อีวานใหญ่ทรงกระดูกงูเรือ (keel-shape) ใหญ่เป็น 2 เท่าอีวานอื่น★ ทางทิศตอ.
- ลานล้อมระเบียงซุ้มโค้งแหลม (pointed arches) + ★เพดานมุก็อรนัสซับซ้อน จบที่ดาว 8 แฉก★ + อาราเบสก์
- ★ทางเข้าตต. แบบ "มาบัยนฺ" (ma bayn)★ — โถงบังตา บล็อกมุมมองไม่ให้เห็นลานใน (privacy สูงสุด)
- ปริศนา: ห้องที่คิดว่าเป็น "มัสยิด" ★ไม่มีมิห์รอบ ไม่มีจารึกกุรอาน เบี่ยงจากกิบลัต 30°★

## 7. สุสานซุมุรรุด คอทูน (Zumurrud Khatun, ~1202, Karkh)
- ★ฐานแปดเหลี่ยม (octagonal base)★ — กระจายน้ำหนัก
- ★โดมทรงกรวยจากมุก็อรนัส 9 ชั้น★ (แต่ละชั้น ~16 เซลล์) — ★เอามุก็อรนัส (ปกติอยู่ใน) มาไว้ "ผิวนอก" → รูปทรงคล้ายลูกไพน์โคน (pinecone)★ (หายากมากในสถาปัตยกรรมอิสลาม)
- ★ในเซลล์มุก็อรนัสเจาะช่องเล็กปิดกระจกหนา → แสงเรืองรอง พร่ามัว ในห้องหลุมศพ (3×7 ม.)★
- มีหออะซานอิฐ = ★เก่าแก่สุดในแบกแดดที่ยังยืนอยู่★
- ⚠️ ★ตัดทิ้ง: ข้ออ้าง "ต้นแบบเพนตากอนสหรัฐ" = spurious ห้ามใส่★

## 8. มัสยิดอัลคุละฟา + หออะซานอิลข่าน (Al-Khulafa)
- เริ่มสร้าง 902 (al-Muktafi) เป็นมัสยิดวัง "al-Qasr" · พังจากมองโกล 1258 · บูรณะ 1272
- ★หออะซานอิลข่าน (~1279): สูง 34–35 ม. อิฐ+ปูน★ · ★ฐาน 12 เหลี่ยม★ · ฐานรับด้วย ★มุก็อรนัส 4 ชั้น★ · เพลาทรงกระบอกสลักลายเรขา + ★อักษรคูฟิก★ (คล้ายหออะซาน Bastam อิหร่าน)
- (เคยเอียงจนชาวบ้านเรียก "ไอ้ค่อม" al-Ahdab · บูรณะยุค 1960 โดย Mohamed Makiya — อิฐเหลืองล้อมหอเก่า)

## 9. คานมัรญาน (Khan Murjan caravanserai, ~1358)
> นวัตกรรม: ★โรงพักแรมที่มี "โถงกลางมีหลังคาคลุมปิด" (covered hall) แทนลานเปิด★ (แปลกจาก khan ทั่วไป)
- ★เพดานโถงสูง 13 ม. (43 ฟุต)★ (~25 ม. จากชั้นล่างสุด)
- ★หลังคา = ซุ้มโค้งยอดแหลม (pointed arch) อิฐมหึมาเรียงจังหวะ + เจาะช่องหน้าต่างโปร่งแสง★ ตลอดแนว → แสงธรรมชาติเข้าโดยไม่ต้องมีลานเปิด
- 2 ชั้น · 45 ห้อง (22 ล่าง + 23 บน) · โถงกลาง = ลานประมูล/ค้าขาย
- ตกแต่งภายนอก = อักษรประดิษฐ์อิสลาม

## 10. ฮัมมัม (โรงอาบน้ำ)
- พัฒนาจาก Roman thermae: ห้องเย็น (apodyterium) → อุ่น (tepidarium) → ร้อน (caldarium) — ปรับให้เข้ากับการชำระล้าง (taharah)
- ★หลังคาโดม "ไม่มีหน้าต่างด้านข้าง" (เก็บไอน้ำ/ความร้อน)★ · ★เจาะช่องแสงกระจกที่ยอดโดม (clerestory / shukhsheikha)★ → แสงทะลุม่านไอน้ำ
- (Hilal al-Sabi อ้าง 60,000 แห่ง = ★hyperbole★ ใช้ back-calc ประชากร)

## 11. ★บ้านพื้นถิ่นแบกแดด (vernacular — ใช้ชื่อ/องค์ประกอบเติมเมือง)★
> นวัตกรรม climate-responsive (passive design) สู้อากาศร้อนแห้ง + privacy
- ★hosh★ = ลานกลางบ้าน ("ปอด" ของบ้าน — ดึงอากาศเย็นกลางคืน, มีต้นไม้/น้ำพุ evaporative cooling)
- ★mejaz★ = ทางเข้าหักมุม (บังตา, privacy)
- ★iwan★ = ซุ้มโถงเปิดด้านหนึ่ง
- ★talar / tarma★ = ระเบียง/แกลเลอรีกึ่งเปิด (tarma หันใต้ รับอุ่นฤดูหนาว)
- ★sardab★ = ห้องใต้ดิน (เย็นคงที่ทั้งปี, มีบ่อน้ำ — ครอบครัวย้ายลงอยู่กลางวันฤดูร้อน)
- ★badgir★ = หอดักลม (จับลมมรสุมตต.เฉียงเหนือ ต้อนลงห้อง/ใต้ดิน)
- ★shanashil / mashrabiya★ = ระเบียงหน้าต่างไม้ฉลุยื่นชั้น 2 (ลดแดด + ลมผ่าน + privacy)
- ★ursi★ = ห้องรับแขกมีประตู-หน้าต่างไม้กระจกสี
- ★บ้านชิดกัน → ซอยแคบลึก = เงาทอดทับ (shading)★ · ผนังอิฐหนา = thermal mass

## 12. วัสดุ + ระบบโครงสร้าง (สรุปยุคอับบาซิด)
- ★วัสดุ:★ อิฐดิบ + อิฐเผา · ปูนปลาสเตอร์/ยิปซัม · หินอ่อน/หินบางส่วน · ★ทาร์ (tar) กันชื้น★
- ★โครงสร้าง:★ ซุ้มโค้ง (arch) · โดม (dome) · vault · ผนังหนา · ลานใน (courtyard) · อีวาน · ฐานราก/ระเบียงยกพื้น
- ★ตอบภูมิอากาศ:★ ผนังหนา · คอร์ต · sardab · badgir · กันชื้นด้วยทาร์

## 13. ★ศิลปะตกแต่ง (decorative vocabulary — หัวใจความเป็นอับบาซิด)★
- ★อิฐเป็นทั้งโครงสร้างและผิวตกแต่ง★ — ฮะซัรบาฟ (hazarbaf): อิฐเรียงลายเรขานูน (ก้างปลา/ตาราง/ซิกแซก)
- ★มุก็อรนัส (muqarnas):★ รังผึ้งหินย้อย — ใต้ซุ้ม/ฐานหอ/เพดาน (ซุมุรรุดเอามาไว้ผิวนอก)
- ★อักษรคูฟิก (Kufic):★ อาหรับเหลี่ยม เป็นแถบประดับ (★ใช้นามธรรม/วลีกลาง — ห้ามแต่งข้อความศาสนาเท็จ★)
- ★ลายเรขา ดาว 8/12 แฉก (girih):★ เหรียญ/แผงข้างซุ้ม
- ★อาราเบสก์ (arabesque):★ พรรณพฤกษาม้วน + ปูนปั้นแกะ (carved stucco, สไตล์ Samarra beveled)
- กระเบื้องเคลือบ turquoise (จุดเน้น เช่น โดมเขียว)
