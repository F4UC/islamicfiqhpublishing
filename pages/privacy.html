Islamic Fiqh Publishing

นโยบายความเป็นส่วนตัว

Islamic Fiqh Publishing ให้ความสำคัญกับความเป็นส่วนตัวของผู้อ่าน เว็บไซต์นี้ออกแบบให้เก็บข้อมูลน้อยที่สุด ไม่มีระบบติดตามรายบุคคล และไม่ส่งข้อมูลของคุณให้บุคคลที่สาม

1. เราเก็บอะไร

เราไม่เก็บข้อมูลส่วนบุคคล เว็บไซต์ไม่มีระบบสมาชิก ไม่ต้องลงทะเบียน และไม่เก็บชื่อ อีเมล หรือหมายเลขโทรศัพท์ของผู้อ่าน

2. คุกกี้และการจัดเก็บในเครื่อง

  • เราไม่ใช้คุกกี้เพื่อการติดตาม (tracking)
  • Cloudflare อาจวางคุกกี้ความปลอดภัยที่จำเป็น (เช่น __cf_bm) เพื่อป้องกันบอทและการโจมตี
  • localStorage เก็บเฉพาะการตั้งค่าการอ่านในเครื่องของคุณเท่านั้น ได้แก่ โหมดมืด/สว่าง ขนาดฟอนต์ และการเปิด/ปิดอักษรอาหรับ ข้อมูลนี้อยู่ในเบราว์เซอร์ของคุณ ไม่ถูกส่งออกไปที่ใด

3. สถิติการเข้าชม

เราใช้ Cloudflare Web Analytics ซึ่งไม่ใช้คุกกี้ และไม่เก็บหมายเลข IP เป็นข้อมูลระบุตัวตน เราเห็นเพียงภาพรวม เช่น บทความที่ได้รับความนิยม ประเทศ และประเภทอุปกรณ์ โดยไม่ติดตามผู้อ่านเป็นรายบุคคล

นอกจากนี้ เรานับยอดเข้าชมบทความแบบนิรนาม โดยไม่เก็บข้อมูลส่วนบุคคล หมายเลข IP หรือคุกกี้ ระบบนับเพียงจำนวนการเข้าชมรวมของแต่ละบทความบนโครงสร้างพื้นฐานของเราเอง เพื่อใช้แสดงบทความยอดนิยมเท่านั้น

4. ฟอนต์และทรัพยากร

  • ฟอนต์ทั้งหมด self-host บนเซิร์ฟเวอร์ของเราเอง ไม่ส่งข้อมูลการเข้าชมไปยังบริการฟอนต์ของบุคคลที่สาม
  • ไม่มี Google Analytics ไม่มี Facebook Pixel และไม่มีเครื่องมือติดตาม (tracker) จากภายนอก
