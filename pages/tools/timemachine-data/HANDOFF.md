## 0. กฎเหล็ก (อ่านก่อนแตะอะไร)
1. **ห้าม merge เข้า `main` เด็ดขาด** (Rule 74 + Push & safety discipline) — push ตรง main ถูกบล็อก
   503 · เอเยนต์ push เพื่อ **รีวิว** เท่านั้น · **One เท่านั้นที่ merge**
2. **อาหรับ byte-exact (Rule 60):** `arabicExcerpt` ทุกชิ้นต้องคัดลอกตรงจากแหล่ง ห้ามพิมพ์เอง/แก้สระ
   — ใช้ `tm_lib.extract()` ที่การันตี substring เสมอ · ห้ามรัน house-style guard ทับ Arabic
3. **ทับศัพท์ล็อกด้วย glossary (Rule 86):** ชื่อบุคคล/สถานที่/ตำราต้องสะกดตรง `canonicalThai`
   ใน`glossary.json` · ชื่อใหม่ = append เข้า glossary ก่อนใช้ · ไม่แน่ใจ → FLAG ไม่เดา (Rule 52/3)
4. **ร้อยแก้วไทยตาม house-style:** ไม่มีราชาศัพท์กับมนุษย์ (Rule 5), ใช้ "แต่/ไม่ใช่" (S2),
   ขีดสั้น "-" (S3), เว้นวรรค อิบนุ/อบู (Rule 52) — คง "ทรง" ได้เฉพาะกับอัลลอฮ์
5. **รันตัวตรวจก่อน push เสมอ:** `PYTHONUTF8=1 python3 scripts/tm_qc.py` ต้องได้ **exit 0 / PASS**

## 4. Pipeline (แก้ไข/เพิ่มเนื้อหา)
```python
# เติมสำนวนข้ามตำราให้เหตุการณ์เดิม
PYTHONUTF8=1 python3 -c "
import sys; sys.path.insert(0,'scripts'); from tm_add_account import add
add(128, [('bidayah-h128-001','ibn-athir',21712,2317,
  'ذكر قتل الحارث بن سريج','ولا يجيز مروان أمان يزيد، فلا آمنه',
  'อิบนุลอะษีร พาดหัวว่า ...')])"
# จากนั้น: sync index.json (ถ้าจำนวน events เปลี่ยน) → รัน tm_qc.py → commit → push
```
- เพิ่มเหตุการณ์ใหม่: คัดลอกโครง event จากชาร์ดเดิม (schema v3) · `arabicExcerpt` ผ่าน `extract()` ·
  `detail` ผ่าน `house_style()` · `persons/places` ต้องอยู่ใน glossary
- **บันทึกความต่างข้ามแหล่ง** ในตัว detail (เช่น ตัวเลขญิซยะฮ์/วันที่/สถานที่เสียชีวิตที่ต่างกัน) — เป็นคุณค่าหลักของหลายสำนวน

## 5. งานที่เหลือ (ทำตามที่เห็นควร)
1. **เติมสำนวนให้ลึกขึ้น** — ปีที่ยังมีสำนวนเดียวบางเหตุการณ์ (ดู `tm_qc.py` ไม่ได้บังคับ) เติมอัลกามิล/
   อัลมุนตะซ็อมที่จับคู่ได้เพิ่ม
2. **สิบฏ์ อิบนุลเญาซีย์ — มิรอาตุซซะมาน** (ลงทะเบียนใน sources.json แล้ว type=annal) → หา Shamela book id
   แล้วเติม accounts (source=`sibt-ibn-al-jawzi`)
3. **ตำราชีวประวัติ (ซิยัร/ฏอบะกอต/ตะฮ์ซีบ/วะฟะยาต)** → เติม `tarajim[]` ระดับบุคคลใน glossary
   (เก็บเลขเล่ม/หน้าได้จากเชิงอรรถของอัลบิดายะฮ์เอง) — มี proof ที่ al-Hasan al-Basri/Ibn Sirin/al-Hajjaj แล้ว
4. **ตรวจคำแปลเชิงความหมาย (Rule 75)** — เทียบ Thai กับ Arabic ทีละสำนวน (งานตรวจทานหลักของรอบนี้)
